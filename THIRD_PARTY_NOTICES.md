# Third-Party Components

This repository includes Windows runtime files used to decrypt legacy Roborock
voice packages:

- `ccrypt.exe` from the ccrypt project:
  <https://ccrypt.sourceforge.net/>
- `cygwin1.dll` from the Cygwin project:
  <https://www.cygwin.com/>

These files remain subject to their respective upstream licenses. The MIT
license in this repository applies to the project source code and does not
replace third-party licenses.

Official Xiaomi voice files are not stored in the repository. The tool
downloads the selected packages directly from Xiaomi servers when requested.
